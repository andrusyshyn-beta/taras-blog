## TL;DR: Головне за 30 секунд

- **Масштаб катастрофи:** 512,000 рядків вихідного коду Claude Code опинилися в публічному доступі через помилку пакування. Це архітектура продукту з річним доходом $2.5 млрд.
- **Швидкість реакції спільноти:** Поки Anthropic розсилала 8,000 DMCA-скарг, розробники встигли зробити тисячі форків та навіть переписати код на Python для захисту від видалення.
- **Криза довіри:** Anthropic позиціонує себе як "безпечна альтернатива OpenAI". Два великих витоки за тиждень ставлять під сумнів здатність компанії захищати критичні розробки перед IPO на $60 млрд.
- **Подарунок конкурентам:** OpenAI, Google та xAI отримали детальну карту архітектури одного з найкращих AI-інструментів для кодингу безкоштовно.

## Хронологія 512,000 рядків коду

Ранок почався з посту дослідника безпеки Chaofan Shou в мережі X. Він помітив посилання на репозиторій з назвою, яка здавалася занадто гучною, щоб бути правдою. Це був повний код Claude Code. 

Далі спрацював ефект доміно:
1. **Тисячі форків за години:** Спільнота GitHub зреагувала моментально. Поки юридичний відділ Anthropic готував запити, код розлетівся по всьому світу.
2. **Битва за видалення:** Anthropic змушена була запустити понад 8,000 DMCA-запитів, намагаючись зупинити поширення. Але інтернет пам'ятає все.
3. **Хід конем від Sigrid Jin:** Корейсько-канадський розробник Sigrid Jin (відома тим, що витратила рекордні 25 млрд токенів Claude Code за рік) зробила неймовірне — переписала весь вихідний код на Python ще до того, як Anthropic встигла видалити оригінал. Тепер юридично це "похідний продукт", на який значно важче накласти DMCA.

## Подвійний удар по репутації: Mythos та Claude Code

Найгірше в цій ситуації не сам факт витоку, а серійність. Це вже **другий витік за тиждень**. За п'ять днів до цього в мережу потрапили 3,000 внутрішніх файлів, що включали деталі розробки "Mythos" — нерелізованої моделі наступного покоління.

Коли компанія з оцінкою в $380 млрд робить одну таку помилку — це прикрий випадок. Коли це стається двічі за тиждень — це системна проблема з культурою безпеки. Для Anthropic це особливо боляче, оскільки весь їхній маркетинг побудований на противазі "некерованому" OpenAI.
