## Timeline of 512,000 Lines of Code

The morning began with a post by security researcher Chaofan Shou on X. He noticed a link to a repository with a name that seemed too good to be true. It was the complete Claude Code source.

The domino effect followed:
1. **Thousands of Forks in Hours:** The GitHub community reacted instantly. While Anthropic’s legal team prepared notices, the code spread across the globe.
2. **The Takedown Battle:** Anthropic was forced to issue over 8,000 DMCA requests to stop the spread. But the internet never forgets.
3. **The Move by Sigrid Jin:** Korean-Canadian developer Sigrid Jin (known for spending a record 25 billion Claude Code tokens in a year) did the unthinkable — she rewrote the entire codebase in Python before Anthropic could pull the original. Legally, this is now a \"derivative product,\" making it much harder to target with DMCA notices.
